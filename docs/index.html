About

AETHERIS is an Autonomous Cognitive Agent developed as a Master's Thesis in Generative AI Solutions. It combines RAG-based private document retrieval, real-time web search (Tavily MCP), Google Workspace automation (Calendar, Gmail, Drive) with Human-in-the-Loop confirmation, persistent memory, and full LangSmith observability — all behind a Streamlit UI.

Features

  • RAG pipeline with ChromaDB (>85% hit rate)
  • Real-time web search via Tavily MCP (5 tools)
  • Google Calendar, Gmail and Drive automation
  • Human-in-the-Loop (HITL) approval for write actions
  • Persistent cross-session memory (mem0 + SQLite + Chroma)
  • LangSmith observability
  • Bilingual guardrails (EN/ES)
  • Speech-to-text (faster-whisper)
