AETHERIS — Privacy Policy

Last updated: May 2026

1. Overview

AETHERIS is an academic project developed as a Master's Thesis (TFM) at EBIS Business Techschool. It is an autonomous cognitive agent that integrates Google Workspace, RAG-based document retrieval, and real-time web search.

2. Data We Access

AETHERIS requests access to the following Google APIs, solely to perform user-requested actions:

  • Google Calendar API — read and create calendar events on behalf of the user.
  • Gmail API — read and send emails on behalf of the user.
  • Google Drive API — read and manage files on behalf of the user.

3. How We Use Your Data

All data accessed via Google APIs is used exclusively to fulfill the user's explicit requests within the AETHERIS chat interface. We do not store, share, or sell any Google account data to third parties.

4. Data Storage

OAuth tokens are stored locally within the application container and are never transmitted to external services other than Google's own OAuth2 endpoints for authentication.

5. Data Retention

No Google account data is retained beyond the active session. OAuth tokens are refreshed automatically and are invalidated when the user revokes access.

6. User Rights

You can revoke AETHERIS's access to your Google account at any time by visiting Google Account Permissions.

7. Contact

This application is an academic project. For any questions, please open an issue on the GitHub repository.
